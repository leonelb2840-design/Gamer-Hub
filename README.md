# 🎮 Gamer Hub - Central de Recursos Pro

¡Bienvenido al **Gamer Hub**, la central definitiva para potenciar tus partidas! Este proyecto reúne herramientas clave y trucos esenciales para tus juegos favoritos en una interfaz optimizada y rápida.

## 🚀 Funcionalidades Principales
* **🌌 Calculadora Nether (Minecraft):** Convierte coordenadas del mundo real al Nether al instante para que nunca pierdas tus portales.
* **🚗 Manual de Trucos GTA:** Acceso rápido a los mejores trucos para **GTA V, San Andreas (SA) y GTA III**. Domina la ciudad en segundos.
* **🧮 Enlace Directo:** Conexión rápida con la **Calculadora Científica Leonel** para cualquier operación extra.
* **📱 PWA Ready:** Ahora instalable como aplicación en tu dispositivo para acceso instantáneo y uso **Offline**.

## 🛠️ Detalles Técnicos
Este Hub está construido con tecnología web moderna para garantizar fluidez:
* **HTML5 & CSS3:** Diseño responsivo con estética Gamer (Neón & Dark Mode).
* **JavaScript:** Lógica de conversión y filtrado de trucos.
* **Service Worker:** Soporte completo para funcionamiento sin conexión a internet.

## 📥 Instalación
1. Accede desde tu navegador.
2. Selecciona "Instalar en la pantalla de inicio".
3. ¡Lleva el Hub en tu bolsillo a todas partes!
